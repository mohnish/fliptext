## FlipText.js

A simple JavaScript component to flip text.

Made with [<3](https://twitter.com/arrowgunz)

### Demo

[http://mohni.sh/fliptext.js/](http://mohni.sh/fliptext.js/)

### Installation
`<script src='fliptext.js'></script>`

### Example

```
flipText("your text")
// => "ʇxǝʇ ɹnoʎ"

flipText("your text", true)
// => "(╯°□°)╯︵ ʇxǝʇ ɹnoʎ"
```
